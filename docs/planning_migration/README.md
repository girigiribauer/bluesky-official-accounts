# 運用体制の見直し計画

## 目的

[Bluesky 公式アカウント移行まとめ](https://bluesky-official-accounts.vercel.app/)の運用を
Notion 中心の参加しづらい体制から **コミュニティ主導** での分散型モデレーションシステムへ移行する。

## 背景

- 2024年2月から運用開始し、アカウントの移行状況を可視化することで一定の成果を達成
- 現在は省力モードで運用中だが、慢性的にチェック側が不足しており、投稿が急に来ると対応できない
- 新たに分野を定義し、互いに異なる分野で独立させ（重複を許容する）それぞれでモデレーションの仕組みを確立したい

## 見直しの方針

- 分野（field）を導入し、旧分類（カテゴリー）を再整理する
- フォーム投稿を自前の Notion API を叩く形から、自前DBに投げる形へ段階的に移行する
- 段階的な権限付与とポイント制度によるモデレーションシステムを構築する
    - 投稿者、承認者、モデレーター、で構成される持続可能な運用体制
    - 全ての操作ログをオープンにして何かあっても戻せるようにする
    - 分野の下に旧来の分類に相当するものを用意し、分野別の分類整理の管理を委譲する
